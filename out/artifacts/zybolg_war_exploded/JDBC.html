一. JDBC介绍

jdbc(java database connectivity)是一门用java语言操作数据库的技术。sun公司(java的原母公司)在JDK中定义了一套接口，然后由各个数据库厂商去实现这些接口，然后封装成驱动jar包。我们使用时实际上用的是这些jar包中的这些实现类操作数据库。

二. JDBC使用

以mysql数据库为例

public class testJDBC {
    public static void main(String[] args) throws Exception{
        //1.导入驱动jar包
        //2.注册驱动
        Class.forName("com.mysql.jdbc.Driver");
        //3.获取connection对象
        Connection con = DriverManager.getConnection("jdbc:mysql://localhost/db1", "root", "123456");
        //4.通过connection对象获取执行sql语句的prepareStatement对象
        PreparedStatement pre = con.prepareStatement("select * from user ");
        //5.执行sql语句
        ResultSet resultSet = pre.executeQuery();
        //6.处理结果
        while(resultSet.next()){
            int id = resultSet.getInt("id");
            String name = resultSet.getString(1);
            System.out.println("id="+id+"  name="+name);
        }
        //7.释放资源
        pre.close();
        con.close();
        resultSet.close();
    }
}

三个重要对象

1.connection

  • 获取connection

    xxxxxxxxxx
    Connection con =    DriverManager.getConnection("jdbc:mysql://localhost/db1","root","123456")

    getConnection()方法接受三个字符串类型参数,分别是：

    1. 所要连接数据库的url，格式： jdbc:mysql://ip地址(域名):端口号/数据库名称

      若是连接本地数据库，则可写为：jdbc:mysql://localhost:3306/数据库名称(端口号默认3306可省略)，或简写为：jdbc:mysql:///数据库名称

    2. 用户名

    3. 密码

  • connection作用

    1. 获取执行sql语句的对象

      • Statement createStatement();
      • preparedStatement prepareStatement(sql);

    2. 管理事务

      • 开启事务： setAutoCommit(boolean autoCommite)
      • 提交事务：commit()
      • 回滚事务：rollback()

2.Statement—执行sql的对象

  • 主要方法

    1. int executeUpdate(sql) //执行insert,update,delete语句，返回影响的行数
    2. ReultSet executeQuery(sql) //执行查询语句，返回一个结果集对象

3.ResultSet

  • 主要方法

    1. boolean next() 游标向下移动一行

    2. getXXX(参数)：获取数据

    3. Xxx：代表数据类型 如： int getInt() , String getString()

      • 参数：

        1. int：代表列的编号,从1开始 如： getString(1)
        2. String：代表列名称。 如： getDouble("balance")

sql注入问题—使用preparedStatement对象

在拼接sql时，有一些sql的特殊关键字参与字符串的拼接。会造成安全性问题

  1. 输入用户随便，输入密码：a' or 'a' = 'a

    sql：select * from user where username = 'fhdsjkf' and password = 'a' or 'a' = 'a'

  2. 解决sql注入问题：使用PreparedStatement对象来解决

  3. 预编译的SQL：参数使用?作为占位符

六. 总结

对JDBC的使用简单总结一下就是：

导入驱动jar包->注册驱动->通过DriverMannager获取Connection对象->通过Connection对象获取PrepareStatement(或启动事务)->再由PrepareStatement对象执行sql
